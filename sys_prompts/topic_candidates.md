규칙:
- 정보제공형과 클릭유도형을 섞는다.
- 제목은 실제 YouTube 제목으로 바로 사용할 수 있어야 한다.
- 과장, 허위, 공포 마케팅을 피한다.
- 건강/금융/법률 관련 주제는 단정적 표현을 피한다.
- 각 항목마다 실제 기획 의도와 대본 요약을 구체적으로 쓴다.
- 위 기존 주제 목록과 같은 주제를 만들지 않는다.
- 제목 문자열만 다르게 바꾸는 것은 금지한다. 같은 질병, 같은 영양소, 같은 생활습관, 같은 문제 상황, 같은 결론이면 중복으로 본다.
- 예를 들어 기존에 비타민 D, 혈압 관리, 뼈 건강을 다뤘다면 표현을 바꿔 같은 소재를 다시 제안하지 않는다.
- 기존 주제와 겹칠 수밖에 없다면 더 좁은 대상, 다른 원인, 다른 실천 상황, 다른 오해를 다루어 명확히 다른 영상으로 만든다.
